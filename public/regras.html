Regras

Diretrizes do sistema

Resumo das regras operacionais para conta, compras, vínculo e suporte dentro do ecossistema Argos RJ.

Conta

Login e identidade

Use sua conta Discord correta para acessar a loja e a dashboard. Dados divergentes podem impedir futuras entregas reais.

Vinculação

ID do jogo obrigatório

O ID informado no dashboard deve ser o mesmo usado no servidor. Compras só devem ser liberadas após confirmação do vínculo.

Compras

Confirmação antes do checkout

Sem login e sem ID confirmado, o sistema bloqueia a conclusão do pedido para reduzir erros de entrega.

Suporte

Pendências e atendimento

Em caso de problema com login, vínculo, caixas ou inventário, use a página de suporte e o Discord oficial.
